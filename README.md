# Quote Over Image

Create beautiful quote images over backgrounds — no signup, no build, hassle-free.

## Access

**Live:** https://daxinc.github.io/quote-over-image-html/

## Usage

Open `index.html` directly in a browser, or serve it locally:

```bash
python3 -m http.server 8080
```

Then open http://localhost:8080.

## Features

- Pick a background image
- Type your quote and adjust font size, alignment
- Download as PNG

## Adding Images

Add entries to `js/images.js` and place files at `img/<id>.png` and `img/<id>-thumbnail.png`.
